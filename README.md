# nadoSdelatIdealno

Краткий проект для экспериментов и изучения возможностей CSS: позиционирование, слои и логические свойства.

## О проекте

Этот репозиторий — небольшой фронтенд-демо, цель которого — безопасная и понятная площадка для экспериментов с CSS: absolute/fixed/relative/sticky, управление порядком слоёв (`z-index`), а также современные логические свойства CSS (margin-inline, padding-block и т.д.). Проект минимален и не содержит бэкенда — только статические ресурсы для визуальных тестов.

## Структура

- `index.html` — точка входа, демонстрационные секции.
- `styles/style.css` — основной файл стилей проекта (основные эксперименты).
- `style.css` — возможный вспомогательный глобальный файл в корне.
- `fonts/`, `images/`, `video/` — медиа и шрифты для примеров.

Откройте `index.html` в браузере или используйте расширение Live Server для быстрой перезагрузки при правках.

## Как использовать

1. Откройте [index.html](index.html) в браузере.
2. Редактируйте `styles/style.css` и наблюдайте поведение блоков при смене `position`, `z-index`, `inset` и логических свойств.
3. Для тестов на разных направлениях текста используйте `direction: rtl` и логические свойствa (`margin-inline-start`, `padding-block-end`).

## Рекомендации по экспериментам

- Проводите изолированные тесты: создайте небольшие секции с фиксированными размерами.
- Меняйте `position` и ставьте `outline`/`background` для наглядности.
- Используйте DevTools для инспекции слоёв и расчёта `z-index`.
